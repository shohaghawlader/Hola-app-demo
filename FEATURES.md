# Voxa 2.0 feature checklist

## Functional in local demo mode
- Accounts, profiles, followers and blocking
- Live/scheduled/private/followers-only room workflows
- Room search, categories, favorites and reminders
- Speaker seats, requests, moderation and role controls
- Chat, image attachment, reply, reaction, pin and delete
- Gifts, coin/earning calculations, promo and recharge simulation
- Missions, XP, levels, streaks and achievements
- Notifications, reporting and safety center
- Withdrawal request and admin review simulation
- Admin users, rooms, reports, economy and content tools
- Cross-tab sync, PWA install and offline app shell

## Optional shared demo mode
- One shared JSON database through Supabase
- Realtime refresh across connected browsers
- Public anonymous demo policies

## Requires production services later
- Actual low-latency voice/audio
- Secure real authentication
- Real payment and withdrawal processing
- Production-grade relational database and RLS
- Push notification provider
- Automated content/audio moderation
